# TeXstudio LaTeX Project

## Entry point

Open and compile `main.tex` in TeXstudio.

Use the green build button or `F5` in TeXstudio. When references or bibliography change, build a few times until the table of contents, figures, tables, and bibliography are updated.

## Structure

- `main.tex`: main document and document settings.
- `bia/`: cover pages.
- `chuong/`: report chapters and front matter content.
- `tham_khao/`: BibTeX bibliography database.
- `tai_nguyen/`: logo và tài nguyên dùng chung.
- `huong_dan/anh_chup/`: TeXstudio setup screenshots.
- `ban_dich/`: generated LaTeX/PDF files moved out of the source root.

## Clean generated files

Generated files such as `.aux`, `.log`, `.toc`, and `.bbl` can be deleted if needed. Keep `.tex`, `.bib`, images, and the final PDF.
